# Login System

A Python login and authentication system developed while learning functions, loops, dictionaries, and file handling.

## Features

* User registration
* User login
* Password validation
* Login attempts limit
* Account lock mechanism
* File-based user storage

## Concepts Practiced

* Functions
* Dictionaries
* File handling
* Loops
* Error handling
* Authentication logic

## Status

Completed and functional. Future improvements may include password hashing and role-based access control.
