# ATM System

A modular ATM system built with Python.

## Features

* PIN authentication
* Withdraw money
* Deposit money
* Balance inquiry
* Transaction history
* Error handling
* Modular design

## Concepts Practiced

* Functions
* Loops
* Conditional statements
* Error handling
* Lists
* Modules
* State management

## Project Structure

ATM/
├── main.py
├── auth.py
├── banking.py
└── history.py

## Status

Completed and functional. Future improvements may include file-based account storage and multiple user support.
