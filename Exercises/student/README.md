# 🎓 Student Management System

A Python console application for managing student records using lists, dictionaries, functions, loops, and error handling.

## 📖 Overview

This project was built as part of my Python learning journey. It demonstrates how to store, search, and manage student information using Python data structures.

The system allows users to add new students, search for existing students, and display all student records through a menu-driven interface.

---

## 🚀 Features

* Add new students
* Search for students by name
* Display all student records
* Store student information using dictionaries
* Manage multiple student records using lists of dictionaries
* Input validation and error handling
* Menu-driven user interface

---
