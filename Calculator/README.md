# Calculator Application

A Python calculator application with history tracking and memory functions.

## Features

* Addition
* Subtraction
* Multiplication
* Division
* Calculation history
* Memory operations (M+, MR, MC)
* Error handling
* Modular design

## Concepts Practiced

* Functions
* Lists
* Modules
* Error handling
* User input validation
* Program state management
